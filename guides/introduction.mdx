## Concepts cles

### Authentification par cle API

Chaque partenaire recoit une cle API unique (format `vk_...`) configuree par l'administrateur de la clinique. Cette cle determine :

- **Les endpoints accessibles** via un systeme de permissions granulaires
- **Le niveau de detail** des reponses (BASIC, STANDARD ou FULL)
- **La limite de requetes** par heure

### Isolation par clinique

Chaque cle API est liee a une clinique. Vous n'accedez qu'aux donnees de la clinique qui vous a fourni la cle.

### Lecture seule

L'API partenaire est principalement en lecture seule. La seule operation d'ecriture disponible est la **creation de rendez-vous**, destinee aux integrations de type booking.
